# How to run the examples

1. Check you have node and npm installed.

```shell
node --version
npm --version
```

2. Install dependencies.

```shell
npm i
```

4. Update app token values.

```js
const SUMSUB_APP_TOKEN = '';
const SUMSUB_SECRET_KEY = '';
```

5. Run the examples.

```shell
node AppTokenJsExample.js
```
